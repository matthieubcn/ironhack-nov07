Anchor Tags - aka LINKS

Links are found in nearly all web pages. Links allow users to click their way from page to page.

HTML Links are hyperlinks, you can clock on a link and jump to another document or another website. This means that links can be either internal(within the document or folder) or they can be external (website, other app)

By default, the linked page or document will be displayed in the current browser window. If we want to change this, we can by using what is known as the target attribute.
