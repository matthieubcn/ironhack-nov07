CSS Media queries son una forma de orientar el navegador según ciertas características, rasgos y preferencias del usuario, y luego aplicar estilos o ejecutar otro código basado en esas cosas. Tal vez las consultas de medios más comunes en el mundo son las que se dirigen a rangos particulares de la ventana gráfica y aplican estilos personalizados, lo que dio origen a la idea del diseño responsivo.

¿Realmente necesitas las CSS Media queries ?

Media queries son una poderosa herramienta en tu caja de herramientas CSS con emocionantes gemas ocultas. Pero si acomodas tu diseño a todas las situaciones posibles acabarás con un código base demasiado complejo de mantener y, como todos sabemos, el CSS es como un cachorro de oso: bonito e inofensivo, pero cuando crezca te comerá vivo.

Existe un libro bastante interesante que se llama Accesibility for Everyone de Laura Kalbag que resume la accesibilidad en una linda analogia.

Un diseñador accesible crearía una puerta grande para que las personas en silla de ruedas pudieran entrar, mientras que un diseñador universal produciría una entrada en la que cabría cualquier persona independientemente de sus capacidades.

Sé que hablar de diseño universal en la web es difícil y casi suena utópico, pero piénsalo, hay alrededor de 150 navegadores diferentes, alrededor de 50 combinaciones diferentes de preferencias de usuario, esto significa que hay al menos 18 millones de casos posibles en los que tu contenido puede ser mostrado.
